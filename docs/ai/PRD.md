# OutPick PRD

## 목적

OutPick의 제품 요구사항을 AI 에이전트가 빠르게 이해하기 위한 문서다.

## 현재 상태

- 확실하지 않음: 이 문서는 아직 전체 제품 기획을 완성본으로 정리한 상태가 아니다.
- 현재 코드 기준 주요 영역은 로그인, 프로필, 채팅, 룩북, 마이페이지로 보인다.
- 상세 제품 목표와 우선순위는 기능별 설계 과정에서 사용자와 논의한 뒤 갱신한다.

## 작성 원칙

- 요구사항, 구현 디테일, 제약 조건, 완료 기준을 분리한다.
- 요청 범위 밖 기능은 추가하지 않는다.
- 확정되지 않은 내용은 확실하지 않음으로 표시한다.
- 기능별 상세 요구사항은 `docs/ai/features/` 또는 `docs/ai/tasks/`에 둔다.
